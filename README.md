# Secure Password & Data Protection System
This project demonstrates a Python-based password security system
using cryptographic techniques.

## Features
- Password strength checking
- Secure password hashing (SHA-256)
- Data encryption & decryption (AES - Fernet)
- Tkinter-based GUI

## Technologies Used
- Python
- Tkinter
- hashlib
- cryptography (Fernet)

## Acknowledgement
This project was developed for academic learning purposes.
AI-assisted tools and online resources were used for guidance
and understanding cryptographic concepts.
